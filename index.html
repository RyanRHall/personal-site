Some of my favorite projects

Repo Vision

npm GitHub

An npm package that allows developers to view an interactive graph of their local git repository. Built with express and vis.js.

Langton's Ant

Live GitHub

An interactive implementation of the Langton's Ant turing machine. Place multiple ants on the grid and watch their behavior evolve. Built with JQuery.

Veriscope

API Extension Presentation

A virus scanner for your news feed. Veriscope is an ongoing project built with the intention of combating the fake news epidemic. This browser extension uses NLP to scan a user's news feed for potentially malicious content.

App Academy JumpStart

Live GitHub

A free, public course that introduces students to the ruby programming language. I built and ran 6 iterations of this course during my time at App Academy.
