Some personal projects:

Ethereum Block Explorer

Live GitHub

A simple block explorer for the Ethereum blockchain


This block explorer, built using Web3, Infura, and React, allows the user to watch as blocks are added to the blockchain, search for blocks, explore transactions and accounts, and - of course - scroll to their heart's content.

CryptoShips

GitHub

A decentralized application for providing scholarships to MOOCs using cryptocurrencies


CryptoShips is an ongoing personal project where users can anonymously sponsor scholarships to courses hosted by MOOC providers like Coursera. CryptoShips is powered by Ethereum smart contracts, and the application's front end runs on React and Drizzle.

News Hound

GitHub Presentation

A virus scanner for your news feed


News Hound was a project spun out of the Debug Politics Hackachon in San Francisco, Nov 2016. The intent was to combat the fake news epidemic by building a browser extension that would continuously survey a user's news feed for malicious content.

Repo Vision

GitHub npm

Visualize the state of your local repository


An npm package that allows developers to view an interactive graph of their local git repository. Built with Express and vis.js.
