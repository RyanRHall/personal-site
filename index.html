Some of my favorite projects:

News Hound

GitHub Presentation

A virus scanner for your news feed. News Hound is an ongoing project built with the intention of combating the fake news epidemic. This browser extension uses IBM's Watson NLP API to scan a user's news feed for potentially malicious content.

Decentralized Medium Clone

GitHub

A functional DAPP built on Ethereum's smart contracts. This application simulates Medium's self-publishing interface, but here readers can choose to support article authors by paying them with Ether. Built with Truffle, Drizzle, and React.

Repo Vision

npm GitHub

An npm package that allows developers to view an interactive graph of their local git repository. Built with express and vis.js.

Langton's Ant

Live GitHub

An interactive implementation of the Langton's Ant turing machine. Place multiple ants on the grid and watch their behavior evolve. Built with JQuery.
